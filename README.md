# Student Performance Predictor

Machine Learning regression project that predicts student academic performance using demographic and study-related features.

## Features

- Data preprocessing and feature encoding
- Ridge Regression model implementation
- Model evaluation using RMSE and R² score
- Flask web interface for prediction

## Model Performance

R² Score: 0.81  
RMSE: 4.34

## Tech Stack

Python  
Pandas  
NumPy  
Scikit-learn  
Flask
